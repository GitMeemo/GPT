
# GPT

GPT Early Access Website is a fully responsive front-end using Next.js and React, with a strong focus on clean code and responsive design.

The project follows modern front-end standards by using reusable Components and layout-based Containers.

![image alt](https://github.com/GitMeemo/GPT/blob/521cef2e174d5f31f06eb2073bcc24bebdf416a4/GPT.png)

## COMPONENTS

-	Navbar – Responsive navigation bar
-	Brand – Brand/partner showcase section
-	Article – Reusable article card component
-	Feature – Modular feature highlight component

## CONTAINERS

-	Header – Responsive hero section with CTA
-	WhatGPT – Overview and introduction section
-	Possibility – Feature expansion section
-	Blog – Responsive blog/articles layout
-	Footer – Responsive footer section

## KEY HIGHLIGHTS

-	Fully responsive across desktop and mobile devices
-	Clean, readable, and maintainable code
-	Reusable and modular components

## TECH STACK

-	HTML
-	CSS
-	React
-	Next.js
